### 문제 설명

<p>지금 여러분이 평화롭게 문제를 보고 있는 사이, 이미 강당 밖에는 극단 원리주의 민초파 테러리스트 김준원이 학교를 점령했다.</p>

<p>입력받은 수열을 3초 안에 오름차순으로 정렬된 상태로 만들지 않으면 강당에 설치해놓은 민초 폭탄이 터진다.</p>

<p>당신은 수열의 어떤 원소 $A_i$를 다른 수 $X$로 바꿀 수 있다.</p>

<p>이 연산에는 1초가 걸린다.</p>

<p><strong>3...</strong></p>

<p><strong>2...</strong></p>

<p><strong>1...</strong></p>

### 입력 

 <p>첫째 줄에 당신이 정렬된 상태로 만들어야 하는 수열의 길이 $N$이 주어진다.</p>

<p>둘째 줄에 수열의 원소들을 나타내는 $N$개의 정수 $A_1, A_2, \cdots , A_N$이 공백으로 구분되어 주어진다.</p>

### 출력 

 <p>3번의 연산 안에 수열을 오름차순으로 정렬된 상태로 만들 수 있으면,</p>

<ul>
	<li>첫째 줄에 <code>YES</code>를 출력한다.</li>
	<li>둘째 줄부터, 수열을 정렬된 상태로 만들 수 있는 방법을 다음과 같은 형태로 출력한다.
	<ul>
		<li>첫째 줄에, 연산 횟수 $K$를 출력한다.</li>
		<li>이후, $K$개의 줄에 걸쳐, 적용할 연산을 순서대로 출력한다.<br>
		구체적으로, 각 줄에 두 정수 $i$와 $X$를 출력한다. 이는, 원소 $A_i$를 $X$로 바꾸는 연산을 의미한다. ($1 \le i \le N$, $1 \leq X \leq 1\,000\,000\,000$)</li>
	</ul>
	</li>
</ul>

<ul>
</ul>

<p>연산의 횟수를 최소화할 필요가 없으며, 가능한 방법이 여럿 있으면 아무 방법이나 하나 출력해도 괜찮다. 같은 원소를 여러 번 수정해도 된다.</p>

<p>3번의 연산 안에 수열을 오름차순으로 정렬된 상태로 만들 수 없으면,</p>

<ul>
	<li>첫째 줄에 <code>NO</code>를 출력한다.</li>
</ul>
